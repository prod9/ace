AI Coding Environment

// what it does

  • Sets up AI coding tools in one command
  • Manages shared skills & conventions via schools
  • Configures LLM backends (Claude Code, OpenCode, Codex)
  • Keeps environments in sync across your team

// quick start

$ cargo install ace
$ ace setup prod9/school # clone a school, write config
$ ace # launch your AI backend

// supported backends

Claude Code OpenCode Codex

// how it works

  1. 1. Resolves which school to use (from ace.toml)
  2. 2. Fetches/updates the school repository
  3. 3. Symlinks skills into your project
  4. 4. Launches the configured backend
