ACE | Schools | Commands | Config

// commands

// core

ace Launch the configured AI backend
ace setup <spec> Clone a school, authenticate services, write config
ace config Print effective configuration
ace paths [key] Print resolved filesystem paths

// skills

ace import <source> Import a skill from an external repository
ace diff Show uncommitted changes in the school cache
ace fmt Format skill files

// school management

ace school init Initialize a new school repository
ace school update Re-fetch all imported skills from sources
ace school propose Branch, commit, push, and open a PR for skill changes

// misc

ace fly 🚀

Pass --porcelain for machine-readable output. Pass flags to the backend with ace -- --flag.
